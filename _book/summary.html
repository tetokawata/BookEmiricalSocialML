6.3 ヒストグラム

  • \(D=Reform\) ごとに以下を表示
  • R
  • Python
Data_R |> 
  mutate(Reform = factor(Reform)) |> 
  ggplot(
    aes(
      x = Size,
      fill = Reform,
      group = Reform
      )
    ) +
  geom_histogram(
    aes(y=..density..),
    alpha=0.5,
    position='identity'
    ) +
  theme_bw()

sns.displot(
  Data_Python, 
  x="Size", 
  hue="Reform", 
  stat="density",
  common_norm=False)

6.4 バランス確認

  • 異なるDグループ間で、Xの分布の違いを確認

  • Xの平均差/Xの標準偏差を報告 (Imbens (2015) などで推奨)

  • R
  • Python
bal.tab(Reform ~ Size + BuildYear + Distance + TradeQ,
        Data_R,
        binary = "std", continuous = "std") |> 
  plot()
